std::string CCmdArgs::argValue ( const std::string &  arg)

Получить дополнительный параметр в аргументе

Например, передан аргумент –ifile="C:/dirname/filename.xml". Вызов функции

argValue("--ifile");

вернет "C:/dirname/filename.xml"
